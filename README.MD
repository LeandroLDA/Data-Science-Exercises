<a name="english"></a>

\## 🇺🇸 English Version



\### 🎯 Objective

To practice and master the fundamental tools of a Data Scientist. These isolated exercises focus on building "muscle memory" for syntax, logical problem-solving, and the practical application of specific libraries in simulated real-world scenarios.



\### 📚 Topics \& Courses Covered So Far

\- \*\*Python Programming:\*\* Control flow, Loops, Dictionaries, User-Defined Functions, Error Handling (`try/except`), and List/Dict Comprehensions.

\- \*\*Data Manipulation (Pandas):\*\* Subsetting, Aggregating (`groupby`), and advanced Data Merging (`merge`, `concat`, `merge\_asof`, `merge\_ordered`).

\- \*\*Exploratory Data Analysis (EDA):\*\* Data validation, handling missing values (Imputation/Dropping), Outlier detection (IQR method), and Feature Engineering (`pd.cut`, `crosstab`).

\- \*\*Data Visualization:\*\* Storytelling, subplots, and customization using \*\*Matplotlib\*\* and \*\*Seaborn\*\* (Relational, Categorical, Distributions, and Heatmaps).

\- \*\*Statistics:\*\* Summary statistics, Probability Distributions (Normal, Binomial, Poisson), and Correlation analysis.



\### 🛠️ How it's organized

Each folder corresponds to a specific course or module from the DataCamp track (e.g., `Exploratory\_Data\_Analysis\_FitTrack`, `Joining\_Data\_Pandas`, etc.).

<a name="português"></a>

\## 🇧🇷 Versão em Português



\### 🎯 Objetivo

Praticar e dominar as ferramentas fundamentais de um Cientista de Dados. Estes exercícios isolados focam em construir "memória muscular" para sintaxe, resolução lógica de problemas e a aplicação prática de bibliotecas específicas em cenários reais simulados.



\### 📚 Tópicos e Cursos Cobertos Até o Momento

\- \*\*Programação em Python:\*\* Estruturas de controle, Loops, Dicionários, Criação de Funções, Tratamento de Erros (`try/except`) e List/Dict Comprehensions.

\- \*\*Manipulação de Dados (Pandas):\*\* Filtragem, Agregação (`groupby`) e Cruzamento avançado de Tabelas (`merge`, `concat`, `merge\_asof`, `merge\_ordered`).

\- \*\*Análise Exploratória de Dados (EDA):\*\* Validação de dados, tratamento de valores nulos (Imputação/Exclusão), detecção de Outliers (método IQR) e Engenharia de Variáveis (`pd.cut`, `crosstab`).

\- \*\*Visualização de Dados:\*\* Storytelling, subplots e customização usando \*\*Matplotlib\*\* e \*\*Seaborn\*\* (Gráficos Relacionais, Categóricos, Distribuições e Mapas de Calor).

\- \*\*Estatística:\*\* Estatística descritiva, Distribuições de Probabilidade (Normal, Binomial, Poisson) e Análise de Correlação.



\### 🛠️ Organização

Cada pasta corresponde a um curso ou módulo específico da trilha da DataCamp (ex: `Análise\_Exploratoria\_FitTrack`, `Juncao\_de\_Dados\_Pandas`, etc.).
